

# 👗👕🛒 Libsy - eCommerce Platform

Libsy is an eCommerce platform for women's clothing and accessories. It is built with ASP.NET Core for the backend and Angular for the frontend. The application incorporates Identity and JWT for authentication and authorization, applies the Unit of Work design pattern and Repository pattern, and follows the MVC architecture in Angular.

## Features

- 🔒 **User Authentication and Authorization**: Uses Identity and JWT for secure login and access control.
- 🛠️ **Admin Dashboard**: Admin users can access the dashboard to manage the platform.
- 🏗️ **CRUD Operations**: Admins can Create, Read, Update, and Delete products.
- 👥 **Role Management**: Admins can assign roles to users to manage access control within the application.
- 🛍️ **Product Management**: Comprehensive management of women's clothing and accessories.
- 🛡️ **Angular Guards**: Protect routes and ensure proper authorization in the frontend.
- 🗃️ **Unit of Work and Repository Pattern**: Implements these design patterns for efficient and organized data management.
- 🌐 **MVC Architecture in Angular**: Follows MVC principles for clean and maintainable frontend code.
